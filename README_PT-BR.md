# Acordo de privacidade

A função "Testar" deste app obterá o status das chaves do sistema correspondentes, incluindo:
- Opções do desenvolvedor
- Depuração USB
- Depuração por Wi-Fi

Este app não coletará nenhuma de suas informações.
